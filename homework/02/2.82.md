# 2.82

```text
/* Cerate some arbitrary values */
int x = random();
int y = random();
/* Convert to unsigned */
unsigned ux = (unsigned) x;
unsigned uy = (unsigned) y;
```

## A. (x<y) == (-x>-y)

当 x == INT_MIN 的时候，-x == INT_MIN ，等号右边不成立

## B. ((x+y)<<4 + y - x) == 17*y + 15*x

表达式，总是成立

因为 17*y == y<<4 + y

因为 15*x == x<<4 - x

整数的乘法满足分配律和结合律

## C. ~x + ~y + 1 == ~(x+y)

表达式，总是成立

~x = -x-1, ~y = -y-1

~x + ~y + 1 = -x - 1 - y - 1 + 1 = -(x+y)-1 = ~(x+y)

## D. (ux - uy) == - (unsigned)(y-x)

表达式，总是成立




## E. ((x>>2)<<2) <= x

表达式，总是成立

令 x' = ((x>>2)<<2) = x - ( 2*x1 + x0)

可得

x' + (2*x1+x0) = x

可知上式左边不可能溢出，所以，x' <= x 总是成立
